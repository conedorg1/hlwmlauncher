h1. hlwmlauncher

A simple app launcher for use with the herbstluftwm tiling window manager

h2. Platform support

The platform is intended to be used on installation of the "herbstluftwm":https://herbstluftwm.org/ tiling WM running on a standard Ubuntu installation.
